# WorkFlow Note product decisions

## Included in MVP
- Korean workplace document generation
- Meeting note / weekly report / daily log flows
- Action item extraction
- Rewrite controls
- History save/load

## Explicitly excluded
- Billing
- Realtime meeting transcription
- Team workspace permissions
- Deep external integrations

## Why this scope
The highest-value user flow is turning rough notes into structured Korean business documents, then making those outputs easy to revise and share. That is the shortest path to user validation.
